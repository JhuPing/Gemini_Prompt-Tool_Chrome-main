# AI 提示詞助手

提升 AI 溝通效率的文字片段工具。
這是一款專為 Gemini 打造的提示詞管理助手，讓您能夠建立專屬指令庫，透過簡單的快捷鍵即刻呼叫複雜的提示詞。

---

## ✨ 核心功能
- **快捷指令呼叫**：在對話框輸入 `/` 即可彈出指令選單。
- **指令庫管理**：專屬後台頁面，輕鬆新增、編輯、刪除您的提示詞範本。
- **個人化視覺**：自由調整選單顏色、字體大小，並提供即時預覽功能。
- **本地化儲存**：所有資料皆儲存在您的瀏覽器本地空間，兼顧隱私與速度。

## 🛠️ 安裝教學
目前本工具支援「開發者模式」手動安裝：
1. 下載本專案原始碼並解壓縮。
2. 開啟 Chrome 瀏覽器，進入 `chrome://extensions/` (擴充功能頁面)。
3. 在右上角開啟 **「開發者模式」** 開關。
4. 點擊 **「載入解壓縮擴充功能」**，選擇本專案的資料夾。
5. 安裝完成後，點擊擴充功能圖示即可進入管理後台。

## 🎨 使用方式
1. 進入後台「提示詞清單」新增您的指令（可設定「名稱」與「代號」）。
2. 至 [Gemini](https://gemini.google.com/) 對話框輸入 `/`。
3. 使用鍵盤 `↑` `↓` 鍵選取指令，按下 `Enter` 自動填入內容。
