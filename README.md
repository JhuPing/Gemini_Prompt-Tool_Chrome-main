# AI 提示詞助手

提升 AI 溝通效率的文字片段工具。
這是一款專為 Gemini 打造的提示詞管理助手，讓您能夠建立專屬指令庫，透過簡單的快捷鍵即刻呼叫複雜的提示詞。

---

## ✨ 核心功能

- **三槽式提示詞系統**：輸入框上方提供「身份」、「指令」、「風格」三個 Tag 槽，各自獨立選取，組合出精準的提示詞前導。
- **智慧搜尋選單**：點擊 Tag 槽彈出搜尋選單，支援關鍵字即時過濾，長文字自動截斷顯示（保留前後內容），搜尋與送出仍使用完整原文。
- **分頁瀏覽**：選單每頁顯示 5 筆，超過時自動出現分頁導覽，避免選單過長難以操作。
- **智慧合併送出**：按下 Enter 後，程式自動將已選取的身份／指令／風格與輸入內容組合，以自然語言格式送出給 Gemini，輸入框全程保持乾淨。
- **中文輸入法相容**：正確處理注音／拼音輸入，選字確認的 Enter 不會誤觸送出。
- **清除記憶**：一鍵送出重置指令給 Gemini，同時清空三個 Tag 槽，方便切換新主題。
- **提示詞庫管理**：後台頁面支援新增、編輯、刪除、上下排序，分「身份」、「指令」、「風格」三個分類獨立管理。
- **個人化外觀**：自由調整選單背景色、主題色、名稱文字色、內容文字色與字體大小，並提供即時預覽。
- **本地化儲存**：所有資料皆儲存在瀏覽器本地空間（chrome.storage.local），兼顧隱私與速度。

## 🛠️ 安裝教學

### Chrome「擴充功能」安裝
1. 前往擴充功能 [下載網址](https://chromewebstore.google.com/detail/ai-%E6%8F%90%E7%A4%BA%E8%A9%9E%E5%8A%A9%E6%89%8B/oihohpgcdckcnhfdigclipjcejhghkme?utm_source=ext_app_menu)
2. 點選「加到 Chrome > 新增擴充功能」

### 「開發者模式」安裝方式
1. 下載本專案原始碼並解壓縮。
2. 開啟 Chrome 瀏覽器，進入 `chrome://extensions/`（擴充功能頁面）。
3. 在右上角開啟 **「開發者模式」** 開關。
4. 點擊 **「載入解壓縮擴充功能」**，選擇本專案的資料夾。
5. 安裝完成後，點擊擴充功能圖示即可進入管理後台。

## 🎨 使用方式

### 建立提示詞庫
1. 點擊擴充功能圖示進入管理後台。
2. 在左側選單選擇「身份」、「指令」或「風格」分類。
3. 點擊右上角「＋ 新增」，填入名稱與提示詞內容後儲存。
4. 可使用 ↑ ↓ 按鈕調整排序，方便日後快速找到常用項目。

### 在 Gemini 使用
1. 前往 [Gemini](https://gemini.google.com/)，輸入框上方會出現三個 Tag 槽。
2. 點擊「身份：未選擇」、「指令：未選擇」、「風格：未選擇」任一 Tag 槽，彈出搜尋選單。
3. 可直接捲動選取，或輸入關鍵字搜尋，找到後點擊選取。
4. 三個槽不需全部選取，只選需要的即可，未選的不會出現在送出內容中。
5. 在輸入框輸入你想問的內容，按下 Enter，工具自動組合並送出。

### 送出格式範例

**使用身份、指令 + 輸入內容：**
```
你是一位【資深翻譯人員】（身份）。
請【將我提供給你的中文內容翻譯成英文】（指令）
---
蘋果（輸入內容）

```

**使用身份、指令、風格 ＋ 輸入內容：**
```
你是一位【資深翻譯人員】（身份）。
請【將我提供給你的中文內容翻譯成英文】（指令）
請以【最精簡的方式】（風格）回應。
---
蘋果（輸入內容）
```

### 清除記憶
- 點擊「清除記憶」按鈕，自動送出重置指令給 Gemini，並同時清空三個 Tag 槽。
- 適合在同一個對話視窗切換完全不同主題時使用。
